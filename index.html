Acra catches exceptions, retrieves lots of context data and send them to a Google Spreadsheet... or whatever backend you prefer.

Best of all, it is FREE and OPEN SOURCE.

Quick setup

Adding ACRA to your project is a matter of minutes. Its impact on your code is minimal. Below are the few steps needed to make your app report its crashes. For more details see the full setup instructions.

Maven dependency

<dependency>
  <groupId>ch.acra</groupId>
  <artifactId>acra</artifactId>
  <version>4.4.0</version>
</dependency>
OR

Manual include

  • Download ACRA
  • Put the .jar in the 
    libs
    subfolder of your Android project
THEN

AndroidManifest.xml

<manifest ...>
  <application ... android:name="MyApplication">
    ...
  </application>
  <uses-permission android:name="android.permission.INTERNET">
  </uses-permission>
</manifest>

Java

import org.acra.*;
import org.acra.annotation.*;

@ReportsCrashes(formKey = "dGVacG0ydVHnaNHjRjVTUTEtb3FPWGc6MQ") 
public class MyApplication extends Application {
  @Override
  public void onCreate() {
    // The following line triggers the initialization of ACRA
    ACRA.init(this);
    super.onCreate();
  }
}
To get your formKey, import the .csv file provided in the package into Google Docs, then create a Form for the resulting spreadsheet. See the full setup instructions.

Other backends include: Bugsense, Zubhium, HockeyApp... Or you can code your own!
