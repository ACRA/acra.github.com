How to Debug

This page describes how to capture logs for ACRA.

Gathering logs#

  1. Enable dev logging by inserting this snippet before ACRA.init:
  • Kotlin
  • Java
ACRA.DEV_LOGGING = true
ACRA.DEV_LOGGING = true;
  1. Create a new logcat filter, with log tag parameter ACRA:
  1. If you want to report a bug or get help, stop your app and clear your logcat. Then start your app and do whatever you want to show, e.g. let it crash. Post all logs using the filter you created in 2.
